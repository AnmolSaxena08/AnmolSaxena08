### 🌡️ Temperature-Based Fan Speed Controller — Arduino
```
  LM35 Sensor → ADC → Arduino → PWM Signal → Fan Motor
  ─────────────────────────────────────────────────────
  Temp < 25°C  →  Fan OFF    (PWM duty = 0%)
  Temp 25–35°C →  Fan SLOW   (PWM duty = 50%)
  Temp > 35°C  →  Fan FULL   (PWM duty = 100%)
```
